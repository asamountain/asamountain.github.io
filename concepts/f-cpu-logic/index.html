Why #define F_CPU?

#define F_CPU 16000000UL
// This line is a "Map" for the compiler.
_delay_ms(1000);
The chip only knows "Beats".

If the heart beats 16,000,000 times a second, then 1ms is 16,000 beats.

Without the top line, the chip doesn't know how many beats make a millisecond!
The Translation
PHYSICAL HEART: 16,000,000 bpm
SOFTWARE MAP: 16,000,000
RESULT: PERFECT TIMING
